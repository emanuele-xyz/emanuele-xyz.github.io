Battling Through The Lands - 2023/09/24

Changelog

  • Implemented Phong lighting model: ambient + diffuse
  • Use blender as rudimentary level editor
    • The scene you see has been made using blender
    • I have written a simple exporter (python script) that takes a blender collection, considers it as a level and writes it to an xml file
    • The exporter uses blender custom properties
    • For now, the light and the camera have been hardcoded
  • Implemented level abstraction as a hierarchy of game objects
    • Each game object has a set of components, such as: transform, model, point light, camera
  • Implemented level loading from xml file
  • Implemented scene system
    • Depending on a game object's components, the game object is handled differently by the scene system
    • Game objects that have transform and model components are treated as simple lit scene objects
    • Game objects that have transform and light components are treated as scene point lights. Only one is supported for now
    • Game objects that have a camera component are treated as scene cameras. Only one camera is supported for now
